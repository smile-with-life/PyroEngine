#### PyroEngine 根目录
``` markdown-tree 
 PyroEngine
	 Document
	 ThirdParty
	 Engine
	 Sandbox
	 Tool
	 Models
	 PyroEngine.sln
	 xmake.lua
```
- **Document**：游戏引擎相关文档目录
- **ThirdParty**：本地包仓库和第三方库目录
- **Engine**：游戏引擎源码目录
- **Sandbox**：引擎沙盒程序目录
- **Tool**：游戏引擎工具目录
- **Models**：模型目录
#### 项目目录
``` markdown-tree 
 Engine
	 Asset
	 Build
	 Config
	 Source
	 xmake.lua
```
- **Asset**：项目资产目录
- **Build**：项目构建目录
- **Config**：配置文件目录
- **Source**：源代码目录
#### Engine 源码目录
``` markdown-tree 
 Source
	 Application
	 Core
	 Launch
	 Module
	 Platform
	 Service
	 pch.h
	 pch.cpp
```
**Application**：应用层
**Core**：核心层
**Launch**：入口层
**Module**：功能层
**Platform**：平台层
**Service**：服务层

#### 模型目录
模型目录用于存放 Python 模型项目，每个模型一个子目录，使用帕斯卡命名法。推荐结构如下：
``` markdown-tree 
 Models
	 RandomMapGeneration
		 Scripts
		 Tests
		 Plugins
		 ModelWeights
		 README.md
```
- **Scripts**：可执行脚本入口
- **Tests**：测试用例
- **Plugins**：插件目录
- **ModelWeights**：模型权重与参数
- **README.md**：模型项目说明
