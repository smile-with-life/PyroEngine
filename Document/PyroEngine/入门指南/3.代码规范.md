在 PyroEngine 中我们会遵循一些基本的代码标准和规范。本篇文章旨在介绍 PyroEngine 目前采用的代码规范。
## 适用范围
适用于所有游戏引擎核心模块、工具链及示例代码的开发。
## 命名规范
#### 类和结构体
采用 AaBb 的格式
```
class CodeStyle;
struct CodeStyle;
```
#### 枚举
采用 AaBb 的格式
```
enum CodeStyle;
enum class CodeStyle;
```
#### 函数与方法
采用 AaBb 的格式
```
void CodeStyle();
bool CodeStyle(int32 value);
```
#### 变量
采用 aaBb 的格式
```
int codeStyle;
float codeStyle;
```
#### 类成员变量
采用 m_aaBb 的格式
```
int m_codeStyle;
float m_codeStyle;
```
#### 结构体成员变量
采用 AaBb 的格式
```
struct Case
{
	int CodeStyle;
}
```
#### 枚举成员变量
采用 AaBb 的格式
```
enum class Case
{
	CodeStyle
}
```
#### 全局对象
采用 GAaBb 的格式
```
Engine GCodeStyle;
Editor* GCodeStyle;
```
#### 全局常量
采用 AA_BB 的格式
```
const int CODE_STYLE;
```
#### 命名空间
采用 AaBb 的格式
```
namespace CodeStyle;
```
#### 私有成员方法
采用 AaBb 的格式
```
class Case
{
private:
	void _CodeStyle()
}
```
