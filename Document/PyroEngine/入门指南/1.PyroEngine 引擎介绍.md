## 概述
**PyroEngine** 是一款高性能、跨平台的 C++ 游戏与应用程序开发引擎，采用模块化架构设计，专为专业级游戏开发、工具创作和服务端应用打造。
**具有以下特点**：
- **极致性能**：利用 C++ 底层优化与多线程渲染，释放硬件潜力。
- **灵活扩展**：支持动态加载渲染/物理模块，适配从独立游戏到 3A 项目的需求。
- **全栈开发**：集成游戏逻辑、编辑器工具、客户端 UI 与服务端开发能力。
- **跨平台兼容**：覆盖 Windows/macOS/Linux/iOS/Android主流平台。
**典型应用场景**：
- 2D/3D游戏开发
- 关卡编辑器设计、动画工具制作以及完整的工具链开发
- 适用于跨平台应用程序的快速开发
- 游戏后端系统搭建和多人联机服务端框架的高效实现
## 开发语言
PyroEngine 采用 **多语言协同开发模式**，平衡性能与开发效率：
####  C++
- **用途**：用于开发引擎核心框架、高性能渲染管线、复杂的物理模拟系统以及精细化的内存管理等关键模块。
- **优势**：零成本抽象、直接硬件访问、多线程优化
#### Python
- **用途**：适用于AI行为树设计、机器学习模型集成与部署、自动化测试脚本编写等任务。
- **优势**：快速迭代、丰富的科学计算库（如 NumPy）、易与 C++ 交互
#### Lua
- **用途**：主要用于构建系统。
- **优势**：轻量级、嵌入方便、语法简单
#### HTML/JS/CSS
- **用途**：应用于编辑器用户界面开发以及各类应用程序的 UI 布局
- **优势**：跨平台兼容性强、生态丰富、支持响应式布局
## 架构设计
#### 分层架构：六层解耦，职责单一
PyroEngine 采用分层架构，确保各模块低耦合、高内聚：
- **Platform 层**：负责封装与抽象底层操作系统及硬件接口，确保系统能够在多种不同的平台环境中稳定运行，提供统一的API接口以屏蔽平台差异，从而提高跨平台兼容性和可移植性，降低上层开发的复杂度。
- **Core 层**：提供包括基础数据结构、高效内存管理机制、常用数学函数库以及算法工具等在内的核心功能组件，构建稳定可靠的基础架构，为上层各个功能模块提供强有力的底层支撑，确保系统整体性能与扩展性。
- **Module 层**：由多个功能相互独立的模块组成，每个模块封装特定功能，支持动态加载、卸载及运行时热插拔，增强了系统的灵活性与可维护性，便于按需组合和功能扩展，适应多样化的应用场景需求。
- **Service 层**：通过组合一个或多个 Module 层的功能模块，封装成面向业务的、易用的服务接口，例如资源管理、输入管理、网络通信等，旨在简化开发流程，提升开发效率，并通过标准化服务降低模块间的耦合度，增强系统整体架构的清晰度。
- **Application 层**：具体实现终端业务逻辑，如游戏玩法、编辑器交互或服务端功能等，该层与引擎核心部分高度解耦，使应用开发更专注于业务实现，便于独立迭代与测试，提升项目的可维护性和灵活性。
- **Launch 层**：封装跨平台的主程序入口函数，负责解析命令行参数、初始化运行环境，选择相应的启动模式，为应用程序提供统一且灵活的启动机制，确保在不同平台上具备一致的启动行为和用户体验。
#### 模块化设计：便于扩展支持
PyroEngine 采用模块化设计，通过将不同功能划分为独立模块，实现了高度的灵活性、可扩展性和开发效率的提升。
- **核心模块**：必需的基础功能模块
- **扩展模块**：按需加载的可选功能模块
- **自定义模块**：用户开发的专用功能模块
## 平台支持
#### Windows
- **版本**：Windows 10 (1909+) / Windows 11
- **架构**：x64, ARM64 (Windows on ARM)
- **特性**：DirectStorage, DirectML, WinRT 支持
- **开发工具**：Visual Studio 2022, Windows SDK

- **发行版**：Ubuntu 20.04+, CentOS 8+, Arch Linux
- **桌面环境**：X11, Wayland
- **包管理**：APT, YUM, Pacman
- **容器支持**：Docker, Podman
#### Mac
- **版本**：Mac 11.0 (Big Sur)+
- **架构**：x64, Apple Silicon (ARM64)
- **特性**：Metal 3, Core Animation, Apple Silicon 优化

- **架构**：ARMv8, x86_64
- **特性**：Vulkan, OpenGL ES 3.2, Android NDK 支持
#### IOS
- **版本**：iOS 14.0+
- **设备**：iPhone, iPad, Apple TV
- **特性**：Metal, ARKit 集成, Game Center
## 图形 API 支持
#### OpenGL
- **版本**：OpenGL 4.6, OpenGL ES 3.2
- **特性**：Compute Shaders, Tessellation, AZDO
- **平台**：跨平台兼容层，用于基础渲染
#### Vulkan
- **版本**：Vulkan 1.2, Vulkan 1.3 (可选)
- **特性**：
    - 光线追踪扩展 (VK_KHR_ray_tracing)
    - 网格着色器 (VK_EXT_mesh_shader)
    - 可变速率着色 (VK_KHR_fragment_shading_rate)
    - 动态渲染 (VK_KHR_dynamic_rendering)
- **平台**：Windows, Linux, Android
#### DirectX12
- **版本**：DirectX 12 Ultimate
- **特性**：
    - DirectX Raytracing (DXR) 1.1
    - Mesh Shaders
    - Sampler Feedback
    - Variable Rate Shading
    - DirectStorage 1.1
- **平台**：Windows 10/11
#### Metal
- **版本**：Metal 3
- **特性**：
    - 光线追踪 (MPSRayTracer)
    - 网格着色器
    - 异步计算
    - Apple Silicon 优化
- **平台**：macOS, iOS
