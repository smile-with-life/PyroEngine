## 源代码分支
PyroEngine 源代码将发布在多个分支
#### 发布分支(release)
每个发布分支均对应一个经过验证的可用版本，这些版本通常包含稳定功能与完善文档。该分支适合用户学习引擎基本架构、核心功能，亦可作为开发基础项目的参考。每个发布分支都会经过严格的代码审查与功能测试，确保达到可用质量水平。
#### 开发分支(develop)
开发分支是 PyroEngine 日常开发活动的核心载体，大部分新功能、性能优化及实验性改进均会优先合并至该分支。因此，该分支通常包含项目最新代码，体现引擎未来发展方向。需注意的是，该分支代码可能未经完整测试，存在潜在错误或不稳定性。若您希望体验最新特性、参与功能测试，或深入了解 PyroEngine 演进动态，可选择使用该分支。
#### 主分支(main)
作为项目中最稳定且最核心的分支，主分支仅包含那些已经经过全面测试、严格验证以及充分优化的最新代码版本。该分支通常由开发分支合并而来，并且所有构建与生成操作仅在主分支下进行。一旦构建流程顺利完成，这些代码即可直接部署到生产环境，用于实际运行。此外，根据项目发布计划，主分支还会定期向发布分支推送新版本，以确保产品能够持续迭代并保持高可用性。
## 下载源码
1. 在 GitHub 上下载源代码
2. 安装 Visual Studio，请参照 设置 Visual Studio 文档，确保你安装了运行 PyroEngine 所需的所有VS组件
3. 在资源管理器中打开下载的源代码文件，并运行 `Setup.bat`
4. 运行 `generate.bat` 为 PyroEngine 创建工程文件
5. 打开 `PyroEngine.sln` 文件将项目加载到 Visual Studio 中，右键点击 PyroEngine 项目并选择编译
