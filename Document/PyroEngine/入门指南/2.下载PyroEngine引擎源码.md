## 从 Github 下载 PyroEngine 引擎源代码

```ad-tip
title:注意
要访问 [PyroEngine 引擎源码仓库](https://github.com/smile-with-life/PyroEngine) ，你必须具备以下条件：
- 拥有 GitHub 账号
- 可以访问 Github 官网
```
PyroEngine 提供完整的 C++ 源代码访问权限，以便于开发者研究、定制、扩展和调试整个引擎，并毫无障碍的完成项目的制作和发布
## 源代码分支
PyroEngine 源代码将发布在多个分支
#### 发布分支
每个发布分支代表一个可用版本，可以用于学习和开发
#### 主分支
大部分活跃的开发工作都发生在主分支上，此分支代表引擎的最新版本。此分支可能存在未经测试的问题，如果你希望测试最新的功能或者希望了解 PyroEngine 的最新开发方向可以使用此分支
## 下载源码
1. 在 GitHub 上下载源代码
2. 安装 Visual Studio，请参照 设置 Visual Studio 文档，确保你安装了运行 PyroEngine 所需的所有VS组件
3. 在资源管理器中打开下载的源代码文件，并运行 `Setup.bat`
4. 运行 `GenerateProjectFile.bat` 为 PyroEngine 创建工程文件
5. 打开 `PyroEngine.sln` 文件将项目加载到 Visual Studio 中，右键点击 PyroEngine 项目并选择编译
